이 문서를 전체 복사(Ctrl+A -> Ctrl+C)하여 엑셀이나 워드에 붙여넣으세요.

1. 전체 프로세스 요약

순서 단계명 주요 작업 비고
1 회원가입 이메일/이름 입력 및 계정 생성 기존 기능 활용
2 견적담기 플랜 및 추가옵션 선택 후 장바구니 담기 고객 수행
3 견적서 발행 관리자가 견적 검토 후 공식 견적서 발송 관리자 수행
4 결제 확인 고객 입금 확인 후 시스템 승인 관리자 수행
5 계약서 발행 공식 계약서 작성 및 발행 관리자 수행
6 계약체결 알림 체결 완료 알림 및 마이페이지 업데이트 시스템 자동

2. 단계별 세부사항 (관리자/고객 상세)

단계 담당자 내용 결과물
견적담기 고객 가격표 섹션에서 플랜 선택 및 요청사항 입력 견적 요청 데이터
견적서 발행 관리자 금액 및 일정 확정 후 시스템에서 견적서 생성 PDF 견적서
결제 확인 관리자 무통장 입금 내역 확인 후 결제 완료 처리 결제 완료 상태
계약서 발행 관리자 최종 계약 조건 입력 및 계약서 파일 생성 PDF 계약서

3. 추가 제안 사항 (추가안)

  • A안: 촬영/제작 상태 추적 - 촬영 중, 편집 중 등 상세 진행 상태 표시
  • B안: 자동 알림 시스템 - 단계별 이메일/알림톡 자동 발송
  • C안: 전자 결제 연동 - 카드 결제 및 실시간 계좌이체 시스템
  • D안: 전자 서명 기능 - 계약서에 온라인 서명 기능 추가

4. 초보자용 프로세스 가이드

고객님: 원하는 패키지를 담고 -> 견적서를 보고 입금하면 -> 계약서가 오고 -> 끝!
관리자님: 신청 내역을 보고 -> 견적서를 보내고 -> 돈 들어온 것 확인하고 -> 계약서를 보내면 -> 끝!
